# Quick Click Builds — Spec Sites

Two spec websites built for San Francisco neighborhood shops, plus a landing page.

- `/` — Landing page (Quick Click Builds portfolio)
- `/russell-lock-key/` — Russell Lock & Key (Outer Sunset locksmith, est. 1978)
- `/mission-shoe-wiz/` — Mission Shoe Wiz (Mid-Market cobbler, est. 1985)

Single-file HTML + CSS + minimal JS. No build step. No frameworks. Hosted on GitHub Pages.
